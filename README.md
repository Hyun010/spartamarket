# spartamarket  
## 프로젝트명 : 우리를 위한 중고거래(스파르타 마켓)
### 프로젝트 소개 : 우리가 아는 'ㄷㄱ' 홈페이지의 백엔드의 CRUD 구현과 프론트 엔드를 구현해본 프로젝트이다.  
### 프로젝트 기간 : 2024.04.12 ~ 2024.04.19(8일)
### 사용 기술 : 
    백엔드(Django, Python) : Django=4.2, Python=3.8
    [Django](https://docs.djangoproject.com/ko/4.2/)
    프론트엔드(HTML,CSS)
    DB(SQLite3)
